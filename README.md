# Application iOS – Gestion des Bénéficiaires (SwiftUI)

Cette application iOS permet de gérer des bénéficiaires et leurs groupes à travers une interface simple, moderne et intuitive.  
Elle a été développée en **SwiftUI**, avec une architecture claire et organisée en plusieurs vues.

---

## Objectifs du Projet

- Afficher la liste des bénéficiaires  
- Ajouter, modifier et supprimer un bénéficiaire  
- Gérer les groupes de bénéficiaires  
- Naviguer facilement entre les différentes vues  
- Offrir une interface fluide et responsive en SwiftUI  

---

## Architecture du Projet

### Fichiers principaux

| Fichier | Rôle |
|--------|------|
| **AccueilView.swift** | Page d’accueil de l’application |
| **ListeBeneficiairesView.swift** | Liste des bénéficiaires |
| **GestionBeneficiaireView.swift** | Ajout / modification d’un bénéficiaire |
| **Groupe.swift** | Modèle représentant un groupe |
| **Beneficiaire.swift** | Modèle représentant un bénéficiaire |
| **DistributionApp.swift** | Point d’entrée de l’application |
| **SplashScreen.swift** | Écran d’introduction |
| **logoApp.png** | Logo de l’application |

> Les fichiers SwiftUI sont organisés pour assurer une navigation claire et une bonne séparation des responsabilités.

---

## Fonctionnalités

- Affichage de la liste des bénéficiaires  
- Ajout d’un nouveau bénéficiaire  
- Modification des informations d’un bénéficiaire  
- Suppression d’un bénéficiaire  
- Gestion des groupes  
- Interface moderne en SwiftUI  
- Navigation fluide entre les écrans
